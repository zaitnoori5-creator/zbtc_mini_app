# zbtc_mini_app

دا یو ساده "mini app" دی چې د ZBTC د بیې ښودلو لپاره جوړ شوی. دا صفحه د CoinGecko عامه API څخه د توکن لټون او بیه ترلاسه کولو لپاره کار اخلي. که توکن په CoinGecko کې ونه موندل شي نو کارونکی کولی شي لاسي بیه داخل کړي.

د GitHub Pages URL (که ریپو عامه وي او workflow ځغلول شوی):

https://zaitnoori5-creator.github.io/zbtc_mini_app/

که تاسو غواړئ زه اضافي فیچرونه اضافه کړم (مثلاً تاریخچه، ګراف، یا د بل API ملاتړ)، نو راته ووایاست.
